# Personal Portfolio Website with AI Chatbot

A modern, responsive portfolio website designed to showcase my professional experience, technical skills, and software development projects. The site includes an AI-powered chatbot that provides visitors with an interactive way to learn more about my background, projects, and expertise.

## 🌐 Live Demo

Visit the portfolio here:

**https://my-portfolio-ten-weld-94.vercel.app/**

---

## 🚀 Features

* Responsive design optimized for desktop, tablet, and mobile devices
* Professional portfolio showcasing projects, skills, and experience
* Interactive AI chatbot for visitor engagement
* Project highlights with descriptions and technologies used
* Clean and modern user interface
* Fast deployment using Vercel
* Easy navigation and user-friendly experience

---

## 🤖 AI Chatbot

The integrated chatbot allows visitors to:

* Learn about my background and experience
* Explore featured projects
* Ask questions about my technical skills
* Receive quick information about my work and achievements
* Engage with the portfolio in a conversational way

## ⚡ Getting Started

### Prerequisites

* Node.js
* npm or yarn

### Installation

Clone the repository:

```bash
git clone https://github.com/your-username/your-repository.git
```

Navigate to the project directory:

```bash
cd your-repository
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
